# TinyKeys
A simple keyboard keystrokes tracker, in lieu of WhatPulse.

Currently planning to support MacOS and Linux.
## Built With
- Keylogger
  - GiacomoLaw/Keylogger
- Database
  - SQLite3
- UI
  - FLTK

## To build on MacOS
1. Ensure you have commandline tools installed
2. Cd to project directory
3. `make lib`
4. `make`
5. `make run`
